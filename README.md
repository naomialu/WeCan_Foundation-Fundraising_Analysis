# WeCan_Foundation-Fundraising_Analysis

Database Structure:
- Donor Table
- Donation Table

SQL Techniques Used:
- GROUP BY
- ORDER BY
- Aggregate functions (AVG, SUM & COUNT)
- Filtering
- Segmentation

Insights Generated:
- Donation trends
- Gender contribution comparison
- High-value donor segments
